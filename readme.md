Each project folder is usually self-contained and may include:

## How To Explore This Repository

1. Start with the project folder you want to inspect.
2. Read its own README if available.
3. Check the backend routes and frontend feature structure.
4. Review environment variables and deployment notes.
5. Follow the flow from authentication to product/cart/payment or AI/chat features.
